1. Pengenalan

Anwarul Bayan ("kami", "aplikasi") menghormati privasi pengguna. Kebijakan ini menjelaskan data yang kami kumpulkan, cara penggunaannya, dan hak Anda terkait data tersebut.

2. Data yang Kami Kumpulkan

2.1 Data yang Anda berikan

  • Informasi akun: nama, email, foto profil saat mendaftar atau masuk dengan Google
  • Data aktivitas ibadah: progres tilawah, zikir, shalat, dan amalan harian (disimpan di perangkat Anda)

2.2 Data yang dikumpulkan otomatis

  • Lokasi: untuk menghitung jadwal shalat berdasarkan koordinat Anda
  • Identifikasi perangkat: untuk notifikasi dan layanan Firebase (Auth, Cloud Messaging)

2.3 Data yang tidak kami kumpulkan

  • Kami tidak menjual data pribadi Anda
  • Kami tidak mengumpulkan data untuk iklan pihak ketiga

3. Penggunaan Data

Data digunakan untuk:

  • Menampilkan jadwal shalat sesuai lokasi
  • Menyimpan dan menyinkronkan progres ibadah
  • Mengirim notifikasi pengingat (tilawah, amalan harian)
  • Memproses login dan keamanan akun
  • Memperbaiki bug dan meningkatkan layanan

4. Layanan Pihak Ketiga

Aplikasi memakai layanan berikut:

  • Google Firebase: autentikasi, database, penyimpanan, notifikasi
  • Google Sign-In: untuk login
  • Layanan lokasi: untuk perhitungan jadwal shalat

Kebijakan privasi mereka berlaku untuk data yang diproses oleh layanan tersebut.

5. Penyimpanan dan Keamanan

  • Data ibadah disimpan di perangkat dan, jika Anda login, di cloud
  • Kami memakai praktik keamanan standar untuk melindungi data
  • Koneksi ke server dienkripsi (HTTPS)

6. Hak Anda

Anda berhak untuk:

  • Mengakses dan memperbarui data pribadi
  • Menghapus akun dan data terkait
  • Menonaktifkan izin lokasi (jadwal shalat akan kurang akurat)
  • Menonaktifkan notifikasi di pengaturan perangkat

7. Anak-anak

Aplikasi tidak ditujukan untuk anak di bawah 13 tahun. Kami tidak sengaja mengumpulkan data dari anak di bawah 13 tahun.

8. Perubahan Kebijakan

Kebijakan ini dapat diubah. Perubahan penting akan diberitahukan melalui aplikasi atau email. Penggunaan berlanjut setelah perubahan dianggap sebagai penerimaan kebijakan baru.
