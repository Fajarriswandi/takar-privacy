Penghapusan Akun & Data — Anwarul Bayan

Terakhir diperbarui: 16 Sya'ban 1447 Hijriyah

Anwarul Bayan adalah aplikasi yang dikembangkan untuk membantu pengguna dalam ibadah harian. Halaman ini menjelaskan cara meminta penghapusan akun dan data terkait Anda.

Langkah-langkah Meminta Penghapusan Akun

  1. Kirim email ke anwarulbayan.id@gmail.com dengan subjek: Permintaan Penghapusan Akun
  2. Sertakan alamat email Google yang Anda gunakan untuk login di aplikasi Anwarul Bayan
  3. Kami akan memverifikasi identitas Anda dan memproses permintaan dalam waktu maksimal 30 hari kerja
  4. Anda akan menerima konfirmasi setelah akun dan data terkait berhasil dihapus

Data yang Dihapus

Setelah permintaan diproses, kami akan menghapus:

  • Akun Firebase Auth: email, nama, foto profil yang terhubung dengan login Google
  • Dokumen pengguna di Firestore: data profil, role, dan metadata terkait
  • Cache lokal (jika Anda mengirim perangkat): data aktivitas ibadah, preferensi, dan tugas kustom yang tersimpan di aplikasi

Data yang Mungkin Disimpan

  • Log server/analytics: data agregat atau anonim mungkin tetap ada untuk periode retensi standar (biasanya hingga 90 hari) untuk keperluan keamanan dan pemecahan masalah
  • Backup: backup sistem mungkin menyimpan salinan data hingga siklus rotasi backup selesai (maksimal 90 hari)

Penghapusan Data Lokal (Tanpa Menghapus Akun)

Jika Anda ingin menghapus data yang tersimpan di perangkat tanpa menghapus akun:

  • Logout: Buka Profil → Logout. Ini akan menghapus cache akun dari perangkat.
  • Hapus data aplikasi: Di pengaturan perangkat (Android: Pengaturan → Aplikasi → Anwarul Bayan → Hapus data), Anda dapat menghapus semua data lokal aplikasi.
  • Uninstall: Menghapus aplikasi akan menghapus semua data yang tersimpan di perangkat. Data di cloud (jika Anda pernah login) tetap ada hingga Anda meminta penghapusan akun melalui email.
