Kebijakan Privasi – Takar

Terakhir diperbarui: Januari 2026

Aplikasi Takar ("kami", "aplikasi") menghormati privasi Anda. Kebijakan ini menjelaskan cara kami menangani data dalam aplikasi.

1. Data yang Kami Simpan

Semua data berikut disimpan hanya di perangkat Anda (lokal), tidak dikirim ke server kami:

  • Data pengeluaran (judul, nominal, jenis rutin/tak terduga, status pembayaran)
  • Pengaturan aplikasi (tema terang/gelap, PIN, preferensi fingerprint)
  • PIN yang Anda set: disimpan dalam bentuk hash di perangkat, tidak dalam bentuk teks asli

2. Layanan AI (Gemini)

Fitur "Analisa Keuangan" dan "Konsultasi AI" menggunakan layanan Google Gemini. Saat Anda memakai fitur tersebut:

  • Ringkasan data pengeluaran Anda (tanpa identitas pribadi) dikirim ke Google untuk menghasilkan analisa dan saran.
  • Pemrosesan mengikuti Kebijakan Privasi Google.
  • Anda bisa memilih tidak memakai fitur AI; data pengeluaran tetap hanya disimpan di perangkat.

3. Izin Aplikasi

  • Internet – Hanya untuk memanggil layanan Gemini saat Anda menggunakan fitur analisa/konsultasi AI.
  • Biometrik (sidik jari / wajah) – Opsional; hanya jika Anda mengaktifkan "Buka dengan fingerprint" untuk membuka kunci aplikasi.

4. Tidak Ada Akun atau Server Kami

Takar tidak memakai login/akun. Kami tidak mengoperasikan server untuk menyimpan data pengeluaran Anda. Data ada di perangkat Anda saja.

5. Penghapusan Data

Menghapus aplikasi dari perangkat akan menghapus semua data pengeluaran dan pengaturan yang tersimpan di perangkat tersebut.

6. Perubahan Kebijakan

Perubahan kebijakan privasi akan dicantumkan di halaman ini dengan tanggal pembaruan. Pemakaian aplikasi setelah pembaruan dianggap sebagai persetujuan Anda.

7. Kontak

Pertanyaan terkait kebijakan privasi dapat dikirim ke pengembang melalui repositori proyek di GitHub atau melalui halaman listing aplikasi di Google Play.
